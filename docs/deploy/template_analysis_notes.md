## 关键结构特征

- **样式**: 主题、Normal、Heading 1/2/3、List Paragraph（仅用于变更履历说明）
- **页脚**: 包含公司版权声明（XXX有限公司版权所有 | 机密信息声明）
- **占位符文本**: 模板中所有正文均为占位说明文字（如"编写目的内容"、"按步骤介绍安装过程并附安装截图"）
- **步骤编号**: 安装过程以"第1步:/第2步:/第3步:"形式占位

## 需替换为 WHUSingularity 内容的关键章节

| 模板章节 | 对应本项目 |
|----------|-----------|
| Oracle数据库安装 | → MySQL 安装 |
| JDK的安装 | → JDK 21 安装 |
| Tomcat的安装 | → Spring Boot 内嵌容器 / 无需独立 Tomcat（或 Docker 部署） |
| WEB应用程序发布 | → 各微服务 Jar 包部署 / Docker Compose |
| 数据库部署 | → MySQL + Flyway 迁移 |
| 系统部署信息汇总 | → Nacos / Redis / RocketMQ / Gateway 等组件信息 |
| 硬件环境 | → 实际服务器配置 |
| 系统管理员平台维护 | → Nacos 控制台 / 运维操作 |
